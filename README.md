# 📖 Overview

CarVision is a Machine Learning project that analyzes automobile data and predicts car prices based on various vehicle characteristics such as engine size, horsepower, fuel type, dimensions, and other technical specifications.

The project includes:

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Data Visualization
- Correlation Analysis
- Machine Learning Modeling
- Price Prediction

This project demonstrates the complete workflow of a real-world machine learning pipeline from raw data to predictive insights.

---

# 🎯 Objectives

- Analyze factors affecting car prices
- Discover relationships between vehicle features and price
- Perform feature engineering for improved model performance
- Build a predictive machine learning model
- Visualize trends and patterns within automotive datasets

# ⚙️ Feature Engineering

Several preprocessing techniques are applied:

### Company Extraction

Company names are extracted from car model names.

### Fuel Economy Calculation

```python
Fuel Economy =
(0.55 × City MPG) +
(0.45 × Highway MPG)
```

### Company Price Categorization

Manufacturers are grouped into:

- Budget
- Medium
- High-End

based on average vehicle prices.

# 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/Payal2805/CarVision.git
```

Navigate to the project folder:

```bash
cd CarVision
```

Install dependencies:

```bash
pip install pandas numpy matplotlib seaborn scikit-learn
```

Run the project:

```bash
python carprediction.py
```
